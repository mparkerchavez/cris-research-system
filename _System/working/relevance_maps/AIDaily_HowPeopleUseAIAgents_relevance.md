## 2. CAPABILITY TRAJECTORY EVIDENCE (3 DPs)

### Model Improvement → Operational Autonomy
- **DP7** — Model capability doubled (Aug-Dec 2025); human interventions per session dropped from 5.4 to 3.3; 39% reduction.
  - **Bridges:** Active Idea #4 (Core Capability Endures), Active Idea #14 (Orchestration Infrastructure as Competitive Layer)
  - **Signals:** Capability gains directly enable operational autonomy; intervention intensity decreases proportionally
  - **Relevance:** HIGH — Quantifies capability-to-autonomy translation; validates core capability endurance thesis

- **DP5** — Long-duration agent autonomy (99.9th percentile) doubled: 25 min (Oct 2025) → 45 min (Jan 2026); consistent model-release-driven gains.
  - **Bridges:** Active Idea #8 (Real-Time vs Legacy Timeline Collision), Active Idea #7 (Agent-Native Development)
  - **Signals:** Temporal compression accelerating; agents extending operational duration; timeline collision deepening
  - **Relevance:** HIGH — Quantifies real-time timeline compression; suggests legacy systems increasingly out-of-phase

- **DP8** — Agents request clarification at 16.4% vs 7.1% human interruption; 2.3x agent-initiated negotiation rate.
  - **Bridges:** Active Idea #1 (Tool-Coworker Duality), Active Idea #2 (Coordination Tax & Autonomy Paradox)
  - **Signals:** Agents actively shape autonomy boundaries; bidirectional negotiation, not unilateral control
  - **Relevance:** MEDIUM — Deepens tool-coworker duality model; quantifies coordination tax bidirectionality

## CROSS-CUTTING INSIGHTS

### Strength: Direct Measurement of Autonomy Dynamics
This source excels at quantifying autonomy progression across three dimensions:
1. **Trust calibration** (20% → 40% approval progression)
2. **Intervention intensity** (5.4 → 3.3 interventions per session)
3. **Autonomy duration** (25 min → 45 min session length)

### Weakness: Limited Infrastructure/Orchestration Evidence
Source focuses on Claude Code as standalone system; minimal evidence on:
- Multi-agent coordination costs
- Cross-domain orchestration patterns
- Legacy system integration friction

### Strategic Tension Revealed
**DP16 + DP5 paradox:** 99.9th percentile sessions doubled (25→45 min) while median remained stable (45 sec) = power users accelerating, mainstream user adoption not translating to longer autonomy horizons. This strengthens bifurcation thesis.
