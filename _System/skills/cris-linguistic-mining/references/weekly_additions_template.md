## Template Usage Notes

1. **Header metrics:** Calculate after all sources processed
2. **Framing Notes:** Write after reviewing all assets—synthesize themes
3. **Source order:** Alphabetical by source name
4. **Category completions:** If category has no assets, state "**None captured**" with optional note
5. **Citations:** MUST include metadata comment after every evidence claim
6. **Cross-reference:** Must match actual source sections—verify counts
7. **Convergence themes:** Only note when 2+ independent sources arrive at similar insight
